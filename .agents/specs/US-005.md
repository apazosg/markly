# US-005 — Asistente de chat y tareas

## User Story

**Como** usuario,
**quiero** preguntar a un asistente sobre mis reuniones y convertir acuerdos en tareas,
**para** consultar y hacer seguimiento sin buscar a mano.

## Use Cases

### US-005-UC-01: Chat [INFERRED]
**Flujo:** `chat_page` permite preguntar por texto o voz; muestra la respuesta del backend (RAG) con las reuniones citadas y chips de tareas detectadas con botón "Añadir".

### US-005-UC-02: Tareas [INFERRED]
**Flujo:** `tasks_page` con tres listas: Sugerencias (confirmar ✓ / descartar ✗), Pendientes y Hechas. Las sugeridas (de reuniones o chat) solo pasan a Pendientes al confirmarlas.
**AC:**
- **US-005-AC-01:** Confirmar una sugerida la pasa a `pending`; descartarla la elimina.
- **US-005-AC-02:** Reprocesar una reunión no toca las tareas ya confirmadas/hechas.

## Functional Requirements

### US-005-FR-001 [INFERRED] — *UC-01*
WHEN el usuario pregunta (texto o voz), THE SYSTEM SHALL enviar la consulta al backend y mostrar respuesta, fuentes y tareas sugeridas.

### US-005-FR-002 [INFERRED] — *AC-01*
WHEN el usuario confirma o descarta una tarea sugerida, THE SYSTEM SHALL actualizar su estado en el backend (`pending` o borrada).
